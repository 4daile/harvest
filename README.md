# ℎ𝑎𝑟𝑣𝑒𝑠𝑡

*un jardin digital — adèle*

---

Ce site est un espace de collecte. Des nouvelles, des poèmes, des articles, des notes en cours — des choses écrites et abandonnées, ou pas encore finies. Pas un blog. Pas un portfolio. Quelque chose entre les deux.

→ [harvest-two-omega.vercel.app](https://harvest-two-omega.vercel.app)

---

## ce qu'on trouve ici

| catégorie | contenu |
|---|---|
| `nouvelles` | textes courts, fictions, fragments |
| `articles` | réflexions, essais |
| `blog` | notes en cours, pensées du moment |

---

## stack

Construit avec [Obsidian](https://obsidian.md) + [Digital Garden plugin](https://github.com/oleeskild/obsidian-digital-garden) + [11ty](https://www.11ty.dev/), déployé sur [Vercel](https://vercel.com).

Les notes sont écrites dans Obsidian et publiées manuellement via le plugin. Le repo sert de pont entre le vault local et le site public.
